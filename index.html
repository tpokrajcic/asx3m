Welcome to asx3m

asx3m is a library that enables de/serialization between ActionScript3 objects and XML.
This ActionScript3 de/serializer is designed to be compatible with Java XStream library, but it's not related to XStream project in any other way.

Usage

Class Asx3mer.as should be used as entry point to asx3m library.
Go to QuickStartpage for a simple usage example.

Dependencies

This project has dependencies with Maashaack library (previously VEGAS).
A list of classes from Maashaack in use (all under 'system.data' namespace):
List, Map, Set, Iterator, ArrayList, HashMap, HashSet, ArrayStack
